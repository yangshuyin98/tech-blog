# 目录和文章操作指南

## 目录结构

```
blog/
├── docs/
│   ├── .vitepress/
│   │   └── config.mts        ← 站点配置（改一次）
│   ├── index.md              ← 首页
│   ├── about/index.md        ← 关于页
│   └── posts/
│       ├── index.md           ← 文章列表页
│       ├── spring-boot-vt.md  ← 文章文件
│       └── ...
├── package.json
└── README.md
```

---

## 一、添加文章

### 步骤

**1. 创建文件**

在 `docs/posts/` 目录下新建 `.md` 文件，文件名用英文小写+短横线：

```
docs/posts/redis-cache-strategy.md
```

**2. 写 Front Matter（文件头）**

```markdown
---
title: 'Redis 缓存策略实战：从穿透到雪崩'
category: database
tags: ['Redis', '缓存']
date: 2026-03-25
readTime: '12 min'
---
```

| 字段 | 必填 | 说明 |
|------|------|------|
| `title` | ✅ | 文章标题 |
| `category` | ✅ | 分类（见下方分类表） |
| `tags` | ❌ | 标签数组 |
| `date` | ✅ | 发布日期，格式 `YYYY-MM-DD` |
| `readTime` | ❌ | 阅读时间 |

**3. 写正文内容**

```markdown
## 第一章

正文内容...

```java
// 代码块自动高亮
public class Demo {}
```
```

**4. 在文章列表页添加链接**

编辑 `docs/posts/index.md`，在对应分类下加一行：

```markdown
- [Redis 缓存策略实战：从穿透到雪崩](/posts/redis-cache-strategy) — 2026-03-25
```

**5. 提交推送**

```bash
cd blog
git add -A
git commit -m "add: Redis 缓存策略实战"
git push
```

Netlify 自动构建部署，无需其他操作。

## 二、删除文章

```bash
# 1. 删除文章文件
rm docs/posts/要删除的文章.md

# 2. 从文章列表页移除对应行
# 编辑 docs/posts/index.md，删除那行链接

# 3. 提交推送
git add -A
git commit -m "remove: 删除 xxx 文章"
git push
```

---

## 三、修改文章

直接编辑 `docs/posts/文章名.md`，然后：

```bash
git add -A
git commit -m "update: 优化 xxx 文章内容"
git push
```

## 四、添加新分类

### 步骤

**1. 在分类表里确认分类是否存在**

可用分类值：

| category 值 | 显示名称 |
|-------------|---------|
| `springboot` | Spring Boot |
| `frontend` | 前端 |
| `backend` | 后端 |
| `linux` | Linux |
| `database` | 数据库 |
| `devops` | DevOps |
| `arch` | 架构 |

如果需要添加新分类（比如 `ai`）：

**2. 编辑 `docs/.vitepress/config.mts`**

在导航栏的 `items` 数组中添加：

```typescript
{ text: 'AI/ML', link: '/posts/?cat=ai' },
```

**3. 编辑 `docs/posts/index.md`**

添加分类标题和文章：

```markdown
## AI/ML

- [文章标题](/posts/article-slug) — 2026-03-25
```

**4. 提交推送**

```bash
git add -A
git commit -m "add: 新增 AI/ML 分类"
git push
```

## 七、写作技巧

### VitePress 支持的特殊语法

**提示框：**

```markdown
::: tip 提示
这是提示内容
:::

::: warning 警告
这是警告内容
:::

::: danger 危险
这是危险提示
:::

::: details 折叠标题
默认折叠的内容
:::
```

**代码高亮：**

````markdown
```java
public class Hello {}
```

```bash
npm install
```

```sql
SELECT * FROM users;
```
````

**表格：**

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据 | 数据 | 数据 |
```

**图片：**

```markdown
![描述](图片URL)
```
