## 为什么用 VIM？

Linux 服务器没有图形界面，不能用记事本。VIM 是 Linux 下最强大的文本编辑器，学会了效率极高。

## 安装 VIM

```bash
yum install -y vim
```

## 三种模式

```
┌─────────────────────────────────────────┐
│              普通模式（默认）              │
│     移动光标、复制粘贴、删除撤销          │
│                                          │
│    按 i/a/o        按 Esc                │
│  ┌───────────┐  ←──────────────────┐    │
│  │  插入模式  │                      │    │
│  │  输入文字   │    按 : 进入         │    │
│  └───────────┘  ─────────────────→  │    │
│                  ┌───────────┐       │    │
│                  │ 末行模式   │       │    │
│                  │ 保存退出   │───────┘    │
│                  │ 搜索替换   │  按 Esc    │
│                  └───────────┘            │
└─────────────────────────────────────────┘
```

| 模式 | 进入方式 | 用途 |
|------|---------|------|
| **普通模式** | 默认/按 Esc | 移动、复制、删除 |
| **插入模式** | 按 `i` `a` `o` | 输入文字 |
| **末行模式** | 按 `:` | 保存、退出、搜索替换 |

## 打开文件

```bash
vim file.txt           # 打开文件
vim +10 file.txt       # 打开并跳到第10行
vim +/keyword file.txt # 打开并高亮显示 keyword
```

## 普通模式操作

### 光标移动

| 按键 | 操作 |
|------|------|
| `h` `j` `k` `l` | 左 下 上 右 |
| `↑` `↓` `←` `→` | 方向键 |
| `数字 + ↑/↓` | 上下移动 N 行 |
| `gg` | 跳到首行 |
| `G` | 跳到末行 |
| `数字 + G` | 跳到第 N 行 |
| `^` 或 `Shift+6` | 跳到行首 |
| `$` 或 `Shift+4` | 跳到行尾 |
| `Ctrl+b` | 向上翻页 |
| `Ctrl+f` | 向下翻页 |

### 复制粘贴删除

| 按键 | 操作 |
|------|------|
| `yy` | 复制当前行 |
| `数字 + yy` | 复制 N 行 |
| `p` | 粘贴到光标后 |
| `P` | 粘贴到光标前 |
| `dd` | 剪切（删除）当前行 |
| `数字 + dd` | 剪切 N 行 |
| `dw` | 删除一个单词 |
| `D` 或 `d$` | 删除光标到行尾 |
| `x` | 删除光标所在字符 |

### 撤销与恢复

| 按键 | 操作 |
|------|------|
| `u` | 撤销 |
| `Ctrl+r` | 恢复（反撤销） |

## 进入插入模式

| 按键 | 操作 |
|------|------|
| `i` | 光标前插入 |
| `I` | 行首插入 |
| `a` | 光标后插入 |
| `A` | 行尾插入 |
| `o` | 下方新建一行插入 |
| `O` | 上方新建一行插入 |

编辑完按 `Esc` 回到普通模式。

## 末行模式操作

按 `:` 进入末行模式，输入命令后回车执行。

### 保存与退出

```vim
:w              保存
:q              退出（未修改时）
:q!             强制退出（不保存）
:wq             保存并退出
:x              保存并退出（同 wq）
:w filename     另存为
```

::: tip 快捷方式
普通模式下按 `ZZ`（大写）= 保存退出，`ZQ` = 不保存退出。
:::

### 搜索

```vim
/keyword        向下搜索 keyword
?keyword        向上搜索 keyword
n               下一个匹配
N               上一个匹配
:nohl           取消高亮
```

### 替换

```vim
:s/old/new/        替换当前行第一个 old → new
:s/old/new/g       替换当前行所有 old → new
:%s/old/new/g      替换全文所有 old → new
:%s/old/new/gc     替换全文，每次确认
:10,20s/old/new/g  替换第10-20行
```

### 其他命令

```vim
:set nu          显示行号
:set nonu        取消行号
:10              跳到第10行
```

## 实战练习

```bash
# 复制一个系统文件来练习
cp /etc/passwd ~/test_vim.txt
vim ~/test_vim.txt
```

练习步骤：
1. 按 `gg` 跳到首行
2. 按 `dd` 删除第一行
3. 按 `p` 粘贴回来
4. 按 `u` 撤销
5. 按 `:set nu` 显示行号
6. 按 `/root` 搜索 root
7. 按 `n` 跳到下一个
8. 按 `:q!` 不保存退出

## VIM 配置

在 `~/.vimrc` 中添加常用配置：

```vim
set nu              " 显示行号
set tabstop=4       " Tab 宽度 4 个空格
set shiftwidth=4    " 自动缩进宽度
set expandtab       " Tab 转空格
set autoindent      " 自动缩进
set cursorline      " 高亮当前行
syntax on           " 语法高亮
```

## 别名机制

在 `~/.bashrc` 中定义别名，简化命令：

```bash
# 编辑别名文件
vim ~/.bashrc

# 添加别名
alias ll='ls -l --color=auto'
alias vi='vim'
alias q='shutdown -h now'

# 立即生效（不用重新登录）
source ~/.bashrc
```
