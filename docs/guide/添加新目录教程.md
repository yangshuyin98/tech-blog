# 添加新目录教程

本教程说明如何在博客中添加新的文章分类目录。

## 如何选择分类方式

| 判断条件 | 使用方式 | 示例 |
|---------|---------|------|
| 目录下有**子文件夹** | 方式1：子文件夹分类 | `posts/`、`guides/` |
| 目录下**只有文件** | 方式2：独立分类 | `springboot/`、`java/` |

**简单来说：有文件夹用方式1，只有文件用方式2**

### 当前目录结构

```
docs/
├── posts/          ← 方式1：有子文件夹 (springboot/, springmvc/, frontend/...)
├── guides/         ← 方式1：有子文件夹 (JavaWeb个人博客系统/, 黑马前端教程/)
├── springboot/     ← 方式2：只有文件
├── java/           ← 方式2：只有文件
├── database/       ← 方式2：只有文件
└── devops/         ← 方式2：只有文件
```

## 方式1：子文件夹分类（如 posts、guides）

适用于文章数量较多、需要按主题细分的场景。

### 目录结构

```
docs/
├── posts/                      # 方式1：子文件夹分类
│   ├── springboot/             # 子分类1
│   │   ├── index.md
│   │   └── *.md
│   ├── springmvc/              # 子分类2
│   │   ├── index.md
│   │   └── *.md
│   └── index.md
└── guides/                     # 方式1：子文件夹分类
    ├── JavaWeb个人博客系统/     # 子分类1
    │   ├── index.md
    │   └── *.md
    ├── 黑马前端教程/            # 子分类2
    │   ├── index.md
    │   └── *.md
    └── index.md
```

### 添加步骤

以添加 `tutorials` 目录（包含 `python` 和 `javascript` 子分类）为例：

#### 第 1 步：创建目录结构

```bash
mkdir -p docs/tutorials/python
mkdir -p docs/tutorials/javascript
```

#### 第 2 步：创建数据加载器

创建 `docs/.vitepress/tutorials.data.mts`：

```typescript
import { createContentLoader } from 'vitepress'

export interface Tutorial {
  title: string
  url: string
  date: string
  category: string
  tags: string[]
  readTime: string
  order: number
}

export declare const data: Tutorial[]

function toString(val: unknown): string {
  if (!val) return ''
  if (val instanceof Date) return val.toISOString().slice(0, 10)
  return String(val)
}

export default createContentLoader('tutorials/**/*.md', {
  transform(raw): Tutorial[] {
    return raw
      .filter(({ url }) => !url.endsWith('/'))
      .map(({ url, frontmatter }) => {
        // 从 URL 中提取分类，如 /tutorials/python/xxx -> python
        const match = url.match(/\/tutorials\/([^/]+)\//)
        const category = match ? match[1] : ''

        return {
          title: frontmatter.title || '未命名',
          url,
          date: toString(frontmatter.date),
          category,
          tags: frontmatter.tags || [],
          readTime: frontmatter.readTime || '',
          order: frontmatter.order ?? 999,
        }
      })
      .sort((a, b) => {
        if (a.order !== b.order) return a.order - b.order
        return b.date.localeCompare(a.date)
      })
  },
})
```

#### 第 3 步：创建列表组件

创建 `docs/.vitepress/theme/components/TutorialsList.vue`：

```vue
<script setup lang="ts">
import { ref, computed, onMounted, watch, nextTick } from 'vue'
import { useRoute } from 'vitepress'
// @ts-ignore: VitePress virtual module
import { data as tutorials } from '../../tutorials.data'

const route = useRoute()

const categoryMap: Record<string, string> = {
  python: 'Python',
  javascript: 'JavaScript',
}

const categoryOrder = Object.keys(categoryMap)

const currentCat = ref('')

function updateCat() {
  const path = window.location.pathname
  const match = path.match(/\/tutorials\/([^/]+)\/?$/)
  if (match && categoryMap[match[1]]) {
    currentCat.value = match[1]
  } else {
    currentCat.value = ''
  }
}

onMounted(() => {
  updateCat()
  window.addEventListener('popstate', updateCat)
  watch(
    () => route.path,
    () => nextTick(updateCat)
  )
})

const hasFilter = computed(() => !!currentCat.value)

const filteredTutorials = computed(() => {
  if (!hasFilter.value) return tutorials
  return tutorials.filter((p: { category: string }) => p.category === currentCat.value)
})

const currentCatName = computed(() => {
  return categoryMap[currentCat.value] || ''
})

const grouped = computed(() => {
  if (hasFilter.value) return []
  return categoryOrder
    .filter(cat => tutorials.some((p: { category: string }) => p.category === cat))
    .map(cat => ({
      key: cat,
      name: categoryMap[cat],
      posts: tutorials.filter((p: { category: string }) => p.category === cat),
    }))
})
</script>

<template>
  <div class="tutorials-list">
    <template v-if="hasFilter">
      <h1>{{ currentCatName }} <Badge type="info" :text="`${filteredTutorials.length} 篇`" /></h1>
      <div class="tutorial-items">
        <a v-for="item in filteredTutorials" :key="item.url" :href="item.url" class="tutorial-item">
          <div class="tutorial-title">{{ item.title }}</div>
          <div v-if="item.readTime" class="tutorial-meta">{{ item.readTime }}</div>
        </a>
      </div>
    </template>

    <template v-else>
      <h1>教程 <Badge type="info" :text="`${tutorials.length} 篇`" /></h1>
      <div v-for="group in grouped" :key="group.key" class="category-group">
        <h2>{{ group.name }}</h2>
        <div class="tutorial-items">
          <a v-for="item in group.posts" :key="item.url" :href="item.url" class="tutorial-item">
            <div class="tutorial-title">{{ item.title }}</div>
            <div v-if="item.readTime" class="tutorial-meta">{{ item.readTime }}</div>
          </a>
        </div>
      </div>
    </template>
  </div>
</template>

<style scoped>
.tutorials-list h1 { margin-bottom: 1.5rem; }
.category-group { margin-bottom: 2rem; }
.category-group h2 { border-bottom: 1px solid var(--vp-c-divider); padding-bottom: 0.5rem; margin-bottom: 1rem; }

.tutorial-items {
  display: flex;
  flex-direction: column;
  gap: 1rem;
}

.tutorial-item {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem 1.25rem;
  border: 1px solid var(--vp-c-divider);
  border-radius: 8px;
  text-decoration: none;

}

.tutorial-item:hover {
  border-color: var(--vp-c-brand-1);
  background: var(--vp-c-bg-soft);
}

.tutorial-title {
  font-weight: 600;
  color: var(--vp-c-text-1);
}

.tutorial-meta {
  font-size: 0.85rem;
  color: var(--vp-c-text-3);
  white-space: nowrap;
}
</style>
```

#### 第 6 步：配置导航和侧边栏

修改 `docs/.vitepress/config.mts`：

```typescript
// 1. 添加分类配置
const tutorialsCategoryMap: Record<string, string> = {
  python: 'Python',
  javascript: 'JavaScript',
}

// 2. 添加获取文章函数
function getTutorialsByCategory(category: string) {
  const dir = resolve(__dirname, '../tutorials', category)
  try {
    const files = readdirSync(dir)
      .filter(f => f.endsWith('.md') && f !== 'index.md')
      .sort()

    return files.map(f => {
      const name = f.replace('.md', '')
      const content = readFileSync(resolve(dir, f), 'utf-8')
      const titleMatch = content.match(/^title:\s*(.+)$/m)
      const title = titleMatch ? titleMatch[1].trim() : name
      return { text: title, link: `/tutorials/${category}/${name}` }
    })
  } catch {
    return []
  }
}

// 3. 添加导航
nav: [
  // ... 其他导航
  {
    text: '教程',
    items: [
      { text: '全部教程', link: '/tutorials/' },
      { text: 'Python', link: '/tutorials/python/' },
      { text: 'JavaScript', link: '/tutorials/javascript/' },
    ],
  },
],

// 4. 添加侧边栏
sidebar: {
  '/tutorials/': [
    {
      text: '教程分类',
      items: Object.entries(tutorialsCategoryMap).map(([key, name]) => ({
        text: name,
        link: `/tutorials/${key}/`,
      })),
    },
  ],
  // 为每个子分类生成侧边栏
  ...Object.fromEntries(
    Object.entries(tutorialsCategoryMap).map(([key, name]) => [
      `/tutorials/${key}/`,
      [
        {
          text: name,
          items: getTutorialsByCategory(key),
        },
        {
          text: '其他教程',
          items: Object.entries(tutorialsCategoryMap)
            .filter(([k]) => k !== key)
            .map(([k, n]) => ({ text: n, link: `/tutorials/${k}/` })),
        },
      ],
    ])
  ),
}
```

## 两种方式对比

| 特性 | 方式1：子文件夹分类 | 方式2：独立分类 |
|------|---------------------|-----------------|
| 适用场景 | 文章数量多、需要细分 | 独立主题、文章较少 |
| 目录结构 | `docs/xxx/子分类/文章.md` | `docs/xxx/文章.md` |
| 数据文件 | 共用一个 `xxx.data.mts` | 独立 `xxx.data.mts` |
| 组件 | 支持分类筛选 | 简单列表 |
| 侧边栏 | 动态生成子分类 | 固定配置 |
| 示例 | `posts/`、`guides/` | `springboot/`、`java/` |

## 添加文章

无论哪种方式，添加文章都**无需修改配置**，只需创建 `.md` 文件：

```bash
# 方式1：在子分类中添加
touch docs/posts/springboot/new-article.md

# 方式2：在独立分类中添加
touch docs/docker/new-article.md
```

文章格式：

```markdown
---
title: 文章标题
tags: ['标签1', '标签2']
date: 2026-04-22
readTime: '10 min'
order: 1
---

# 文章标题

内容...
```

侧边栏和列表会自动更新！
